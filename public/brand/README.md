# Brand Asset Gereksinimleri

Gerçek kurumsal logo dosyaları henüz projeye eklenmemiştir. Bu klasöre tasarım ekibi tarafından sağlanacak final assetler konulmalıdır.

Beklenen dosyalar:

- `logo.svg`
- `logo-white.svg`
- `mark.svg`
- `favicon.svg`
- `social-preview.jpg` veya `social-preview.png`
- `brand-pattern.svg`

Notlar:

- PDF veya ekran görüntüsünden düşük kaliteli kırpım kullanılmamalıdır.
- Açık zemin için renkli logo, koyu zemin için beyaz/negatif logo tercih edilmelidir.
- Logo oranları korunmalı, web içinde efekt/gölge uygulanmamalıdır.
