# Kurumsal Font Dosyaları

Bu klasör Okyanus İnsani Yardım Derneği kurumsal fontlarının proje içi kaynağıdır.

Beklenen dosyalar:

- `Gilroy-Bold.woff2`
- `Gilroy-Black.woff2`

TTF fallback kabul edilir:

- `Gilroy-Bold.ttf`
- `Gilroy-Black.ttf`

Font dosyaları lisanslı ve kullanıma uygun olmalıdır. Dosyalar yoksa site ve PDF üretimi güvenli fallback fontlarla çalışmaya devam eder.
